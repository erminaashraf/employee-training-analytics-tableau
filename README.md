## Project Objective
This project supports leadership decision-making at a fictional company (Meadow Park Tech Solutions) by answering:

1. Which training programs are most effective?
2. Which departments deliver the highest ROI on training spend?

---

## Key Techniques Used
- Data cleaning and validation (outliers, duplicates, standardization)
- Calculated fields in Tableau (LOD expressions, performance gain %, cost-efficiency score)
- Filters for analytical rigor (high engagement only)
- Scatterplots, bar charts, annotations, reference lines
- Business-focused dashboard storytelling

---
